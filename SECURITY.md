# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

The project is not actively maintain but we are always ready to publish a security patch. You can send it by mail to julien@duponchelle.info
